# Проект: Автоматизация тестирования с использованием Selenium

## Описание
Этот проект представляет собой набор автоматизированных тестов для сайта www.saucedemo.com,
выполненных с использованием Selenium WebDriver и pytest. Тесты проверяют различные сценарии
поведения пользователей с разными уровнями прав и производительности. Проект помогает убедиться в корректной
работе таких функций, как авторизация, добавление товаров в корзину, оформление покупки, а также специфические ошибки
и визуальные баги.

## Структура проекта

- **`conftest.py`**: файл содержит фикстуру `driver`, которая используется для инициализации и завершения работы веб-драйвера **Chrome**.
- **`test_error_user.py`**: тесты для проверки функциональности при авторизации под пользователем `error_user`.
- **`test_locked_user.py`**: тесты для проверки недоступности функций сайта для пользователя `locked_out_user`.
- **`test_performance_glitch_user.py`**: тесты для проверки производительности сайта при авторизации под пользователем с медленной загрузкой элементов.
- **`test_problem_user.py`**: тесты для проверки багов на сайте при работе с пользователем `problem_user`.
- **`test_standard_user.py`**: тесты для проверки базовой функциональности сайта под стандартным пользователем `standard_user`.
- **`test_visual_user.py`**: тесты для проверки визуальных изменений и багов при работе с пользователем `visual_user`.

## Установка и запуск

### Требования

- Python 3.x
- Установленные библиотеки:
  - `selenium`
  - `pytest`
  - `webdriver-manager`

### Установка зависимостей

Для установки необходимых библиотек выполните следующую команду:

```bash
pip install -r requirements.txt
```

### Запуск тестов
Для выполнения тестов используйте команду:

```bash
pytest
```

Эта команда выполнит все тесты из файлов проекта и выведет результаты тестирования в консоль.

### Описание тестов
Каждый тестовый файл отвечает за сценарии тестирования отдельных пользователей с различными учетными данными. Ниже приведены краткие описания тестов:

### test_error_user.py
- `test_add_all_products_in_basket_and_remove`: Тест проверяет добавление всех товаров в корзину и их последующее удаление.
- `test_complete_purchase`: Тест на завершение покупки с проверкой на ошибки в заполнении формы.
### test_locked_user.py
- `test_authorization`: Тест проверяет недоступность корзины и каталога товаров для пользователя locked_out_user.
- `test_performance_glitch_user`.py
- `test_authorization`: Тест на авторизацию с замедленной производительностью.
- `test_add_items_to_cart`: Тест на добавление товаров в корзину.
- `test_remove_items_from_cart`: Тест на удаление товаров из корзины.
- `test_complete_purchase`: Тест завершения покупки.
### test_problem_user.py
- `test_identical_images_by_products`: Тест проверяет наличие одинаковых изображений товаров.
- `test_add_all_products_in_basket_and_remove`: Тест на добавление всех товаров в корзину и их последующее удаление.
### test_standard_user.py
- `test_purchase_backpack`: Тест на успешную покупку рюкзака.
- `test_view_products`: Тест на просмотр каталога товаров.
- `test_add_many_products_in_basket`: Тест на добавление нескольких товаров в корзину.
- `test_delete_product_before_purchase_from_basket`: Тест на удаление товара из корзины перед покупкой.
### test_visual_user.py
- `test_login_and_price_change`: Тест на проверку изменения цен на товары после перезагрузки страницы.
- `test_cart_icon_position`: Тест на некорректное положение иконки корзины.
- `test_checkout_button_position`: Тест на некорректное положение кнопки Checkout.

## Примечания
Этот проект ориентирован на работу с браузером Google Chrome, но вы можете изменить браузер на другой, скорректировав фикстуру driver в conftest.py.
